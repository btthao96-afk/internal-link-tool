🔗 Semantic Internal Linker

Skill tự động phân bổ internal link cho website dựa trên semantic clustering, graph & anchor distribution.
Skill 1/3 5 bước pipeline Python · sklearn · networkx Source on GitHub

Pipeline

1
Semantic Clustering — gom URL theo chủ đề (KMeans trên embedding)
2
Semantic Graph — graph có hướng + PageRank (networkx)
3
Link Allocation — mỗi URL có 3 OUT + 3 IN, không self-link, không vòng tròn
4
Anchor Distribution — quota theo ratio, anti over-optimization
5
Link Placement — intro / body / outro theo context
