  Internal Link Tool — Công cụ gợi ý internal link cho SEO

  Repo chứa 2 phiên bản độc lập giải quyết cùng 1 bài toán: tự động đề xuất
  internal link giữa các trang trên website để tối ưu SEO.

  🅰️ Local Tool (backend/ + dashboard/) — phiên bản "đầy đủ" tự động

  Workflow CLI 4 bước (backend/scripts/workflow.js):

  1. Crawl (crawler.js) — Dùng axios + cheerio quét toàn site từ URL gốc, chỉ đi
   cùng domain, lấy title, h1, meta, content, headers, lưu vào PostgreSQL.
  2. NLP (nlp.js) — Tokenize Unicode (hỗ trợ tiếng Việt với regex \p{L}\p{N}),
  loại stopwords, tính TF-IDF chuẩn (tf × log(N/df)) trên toàn corpus rồi lấy
  top keywords mỗi trang.
  3. Relevance (relevanceEngine.js) — So sánh keyword giữa các trang (Jaccard
  similarity), xếp hạng và lọc ngưỡng RELEVANCE_THRESHOLD = 70%.
  4. Suggestions (suggestionGenerator.js) — Cho mỗi trang sinh anchor text (từ
  title/H1/keyword + descriptor), tìm vị trí đoạn văn phù hợp, lưu bảng
  suggestions.

  Dashboard React (port 3000) cho duyệt → approve/reject → export. REST API
  Express (port 3001) phơi /api/start, /api/process, /api/pending,
  /api/suggestions/:id/approve…

  🅱️ Web App (web-app/) — phiên bản đa người dùng + tool client-side

  - Backend (Express + JWT, port 5000): login/register với bcrypt, schema 11
  bảng có thêm users, user_sessions, projects (mọi data tách theo project_id).
  - Frontend (InternalLinkSeoTool.js, 1035 dòng) — đây là tool chính đang chạy 
  live trên GitHub Pages. Khác với phiên bản local, nó không cần crawl/NLP mà
  chạy hoàn toàn ở client:
    - User định nghĩa mapping: keywords[] → target URL → anchor texts[] (mỗi
  anchor có thể đặt % phân phối, ví dụ ghế văn phòng | 50).
    - User paste danh sách source URLs.
    - Tool normalize URL, dùng slug (path + query) làm "haystack", match keyword
   không dấu (removeDiacritics), tính score 40 + matchedKw*20.
    - Phân phối anchor theo % chính xác bằng thuật toán buildQuotas
  (largest-remainder rounding).
    - Có fallback: URL nào không match mapping nào → tự tìm overlap keyword giữa
   các URL với nhau, lấy anchor từ "fallback pool".
    - Cho edit từng anchor (override), export CSV, import/export mapping JSON,
  lưu mapping vào localStorage.
