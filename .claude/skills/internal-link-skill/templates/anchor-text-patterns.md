# Anchor Text Patterns

Các mẫu anchor text được khuyến nghị cho internal linking, sắp xếp theo mức độ ưu tiên SEO.

## 1. Exact Match (Dùng dè dặt, max 1-2 lần/bài)
- `[từ khóa chính]`
- Ví dụ: `[internal link SEO]`, `[tối ưu on-page]`

## 2. Partial Match (Khuyến nghị, dùng nhiều)
- `[hướng dẫn từ khóa chính]`
- `[cách làm từ khóa chính]`
- Ví dụ: `[hướng dẫn tối ưu internal link]`, `[cách audit SEO on-page]`

## 3. Branded + Keyword
- `[Brand Name - Keyword]`
- Ví dụ: `[SEONGON hướng dẫn internal link]`

## 4. Long-tail / Natural Language (An toàn nhất)
- Cụm 4-7 từ chứa keyword
- Ví dụ: `[chi tiết về cách xây dựng cấu trúc internal link hiệu quả]`

## 5. LSI / Semantic Keywords
- Anchor text dùng từ đồng nghĩa, từ liên quan
- Ví dụ với keyword "internal link": "liên kết nội bộ", "link nội site"

## Anchor Text CẦN TRÁNH

| Anchor | Lý do |
|--------|-------|
| "click here", "tại đây" | Không có giá trị ngữ nghĩa |
| "read more", "xem thêm" | Generic, không SEO-friendly |
| URL trần (https://...) | Không tự nhiên |
| Anchor lặp lại 100% | Bị Google đánh giá over-optimization |
| Anchor không liên quan tới trang đích | Misleading user |

## Tỷ lệ phân bổ khuyến nghị (cho 1 trang đích)

- 10-20% Exact Match
- 30-40% Partial Match
- 20-30% Long-tail
- 10-20% Branded
- 10% LSI/Semantic
